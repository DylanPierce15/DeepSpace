# Style ID
clean-light

# Style Name
Clean Light

# Style Prompt
Create a minimalist Clean Light UI. Use a pure white background with extensive whitespace/padding. Use a sans-serif font family (like Inter or Helvetica) with varying weights for hierarchy. Borders should be extremely subtle (1px solid #f0f0f0) or non-existent. Shadows should be large, diffuse, and very transparent to create depth without heaviness. Use a single vibrant accent color for primary buttons, but keep the rest of the UI monochrome. Inputs should have ample padding and rounded corners.
